== Description ==

CC Manga Comic Reader help add manga with multi chapter link, can add custom field, custom taxonomy for manga.

== Docs ==
1. After install you can go Dashboard -> Manga to create new manga.
2. To create chapter for manga, you can go Dashboard -> Chapter to add new chapter, when you add chapter to manga, at manga will auto add chapter.
3. To add new custom taxonomy or add new custom field for manga, you can go Dasboard -> Manga -> Settings.
